# Hyprconf-v2 Cheatsheet

## Launcher and Apps
- SUPER + Return: open terminal (kitty)
- SUPER + Shift + Return: open floating terminal (kitty)
- SUPER + Ctrl + T: open terminal (kitty)
- SUPER + D: app menu (rofi)
- SUPER + E: file manager (Dolphin)
- SUPER + Shift + E: terminal file manager (Yazi)
- SUPER + C: VS Code
- SUPER + B: browser (default)

## Window Management
- SUPER + Q: close active window
- SUPER + V: toggle floating
- SUPER + F: toggle fullscreen
- SUPER + Shift + P: toggle pseudo-tiling
- SUPER + M: split ratio 0.3
- ALT + Tab: cycle windows

## System and UI
- SUPER + Shift + M: exit Hyprland
- SUPER + Shift + L: lockscreen (hyprlock)
- SUPER + Alt + L: lockscreen theme
- SUPER + Ctrl + R: reload Hyprland
- SUPER + Ctrl + W: switch Waybar layout
- Ctrl + Esc: reload Waybar
- Ctrl + Alt + Esc: toggle Waybar visibility

## Clipboard and Emoji
- SUPER + Alt + C: clipboard manager
- SUPER + Alt + W: wipe clipboard history
- SUPER + Shift + D: emoji picker

## Screenshot and Wallpaper
- Print: screenshot
- SUPER + W: wallpaper
- SUPER + Shift + W: wallpaper select (thm1)
- SUPER + Ctrl + Shift + W: wallpaper select (thm2)

## Theme and Colors
- SUPER + T: theme selector
- SUPER + Ctrl + P: regenerate colors

## Audio and Brightness
- F9: mute audio
- F10: volume down
- F11: volume up
- SUPER + F1: nightlight down
- SUPER + F2: nightlight up
- SUPER + F3: nightlight reset
- F4: brightness up
- F3: brightness down

## Workspaces
- SUPER + 1..0: switch workspace 1..10
- SUPER + Shift + 1..0: move window to workspace 1..10
- SUPER + Alt + 1..0: move window silently to workspace 1..10
- SUPER + mouse scroll: cycle workspaces

## Focus and Move
- SUPER + H/J/K/L: move focus left/down/up/right
- SUPER + Ctrl + H/J/K/L: move window left/down/up/right
- SUPER + left click: move window
- SUPER + right click: resize window
